# Task Manager (Full Stack Project)

## 🚀 Features Completed
- Create task ✅
- View tasks ✅
- Update task status ✅
- Delete task ✅
- Audit logs tracking ✅
- PostgreSQL integration ✅
- REST API using Express ✅
- React frontend with Tailwind UI ✅

## ⚠️ Partially Completed / Pending
- Authentication (JWT integration) ❌ (currently disabled for testing)
- User login/signup UI ❌
- Multi-user support ❌

## 🛠️ Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL

## ▶️ How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev

## 📌 Notes
- Authentication middleware is temporarily disabled to demonstrate core functionality.
- Dummy user values are used (userId = 1, orgId = 1).
